Class ve ID Kullanimi

Web sayfamizi olustururken HTML elementlerimize bazi stil ozellikleri eklemek isteriz. Bazi yazilarin renkli, bazi resimlerin kucuk veya bazi butonlarin farkli sekilde olmasi gerekebilir ve biz de bunun icin CSS kullaniriz.
Asagide html elementlerine nasil stil ozellikleri ekledigimize bakalim.

Sonuc:

Yukarida goruldugu uzere iki farkli p elementimize kirmizi renk ozelligi eklemis olduk. Fakat sadece belirli p elementine istedigimiz herhangi bir ozelligi eklemek istesek ne yapacagiz? bu durumda class veya id secicilerini kullanmamiz gerekiyor.

class Kullanimi

Class secicisi,HTML uzerinde ayni class'a sahip elemana ulasmamizi saglar.
Class secicisi CSS'de .(Nokta) ile belirtilir.

Yukarida uc adet h4 elementimiz bulunuyor. Bunlardan sadece ikincisine kirmizi renk ozelligi eklemek istersek class secicisini kullanabiliriz.

Daha sonra bunu istedigimiz ozellige ekleyelim.

Sonuc:

bir class'i birden fazla HTML elementi icin kullanabiliriz.

Sonuc:

Eger bir HTML elementinin birden fazla class ozelligine sahip olmasini istiyorsak ayni anda iki farkli class'i kullanabiliriz. Bunun icin sadece iki class arasina bosluk birakmak yeterli olacaktir.

Sonuc:

bir HTML elementi kendini kapsayan elementin (parent elementi) stil ozelliklerine sahip olur.

Sonuc:

Yukarida div elementine CSS ozelligi ekledik fakat altindaki elementler (child elementleri) de bu ozellige sahip oldu.

id Kullanimi

ID secicisi, HTML uzerinde ayni id'ye sahip elemana ulasmamizi saglar.
ID Secicisi CSS'de # ile belirtilir.
ID Secicisinin Kullanim amaci olarak class secicisinden bir farki yok diyebiliriz. Ikisi de belirli HTML elementlerine CSS ozellikleri eklemeye yoneliktir. Fakat id secicisinin class secicisinden bazi farklari vardir.

Bir id'yi sadece bir HTML elementi uzerinde kullanabiliriz.

Asagidaki yanlis bir kullanimdir!
