Kisaca CSS kodlarimizi nasil yazabilecegimizi ogrendikten sonra simdi CSS secicileri kullanarak nasil HTML taglarina erisebibliyoruz onlara bakalim.

Seciciler

Genel Secici

Bu seciciyi kullanarak tum Etiketlere CSS uygula demis oluruz.

ilk kullanimda tum elementlerin margin ve padding degerlerini sifirlamis olduk. ikinci kullanimda div * diyerek tum div elementleri icindeki elementlerin yazi rengine erismis olduk.

Element Seciciler (Element Selectors)

Bu seciciler ile dogrudan HTML etiket isimlerini kullanarak CSS uygulayabiliriz.

Tum div etiketleri etkilenir.

Sinif Seciciler (Class Selectors)

Bu seciciler ile sinif atadigimiz etiketlere CSS uygulayabiliriz. Sinif ismine erismek icin sinif isminin basina nokta (.) ekliyoruz.

Burada ikinci kullanimda p.mavi sinif ismi mavi olan p elementlerine erismis olduk.

ID Seciciler (Id Selectors)

Bu seciciler ile id atadigimiz elementlere CSS uygulayabiliriz. id'ler tek bir elemente ait olmalidirlar. Id ozelligine erismek icin id'nin basina # ekliyoruz.

Ozellik Seciciler (Attribute Selectors)

Bu seciciler ile ozelligi belirttigimiz elementlere CSS uygulayabiliriz. Ozelligin ici bos olsada element bundan etkilenecektir. Ozelliklere erismek icin yapmamiz gereken tek sey koseli parantezler icinde ozelligin ismini [attribute] seklinde yaziyoruz.

Bu sekilde name attribute alan tum elementler etkilenir.

Burada sinifi .btn ve niteligi(attribute) [disabled] olan butona CSS uyguladik.

Burada tam eslesen ozellige CSS uyguladik. Buyuk-kucuk harf duyarliligi vardir.

Burada ~= ifadesi ile title ozelligi"isim" iceren divlere eristik.

Burada ^= ifadesi ile href ozelligi "https" ile baslayan a etiketlerine eristik.

Burada *= ifadesi ile href ozelligi "http" iceren a etiketlerine eristik.

Burada $= ifadesi ile class ozelligi sonunda "test" iceren divlere eristik.

Burada https ile baslayan ve sonunda com olan a etiketlerine eristik.
Artik ana hatlariyla ogrendigimiz CSS secicilere daha yakindan bakabiliriz.

Grup Seciciler (Group Selectors)

Cogunlukla etiketlere verilen CSS ozellikleri benzer veya ortak olabilir. Bu gibi durumlarda secicileri gruplayarak daha temiz CSS dosyalari olusturabiliriz.

Tum h1,h2, h3 etiketlerine erismis olduk.

Burada sinif ismi .turuncu olan p etiketlerine ulastik.

Cocuk Seciciler

Artik etiketleri birbiriyle olan hiyerarsik duruma gore secerek CSS ozellikleri belirleyecegiz. Child Selector, kendisi ve kendisini sarmalayan bir ust etiketle olan iliskiyi gosterir ve > isareti ile ifade edilir.

Parent yani bir ust kapsayici p olan span etiketine ulastik.

Torun-Soy Seciciler (Descentad Selectors)

Bir kapsayici parent element altindaki tum etiketlere ulasmak icin kullanilir. Her ulasilacak etiket arasina bosluk konulur.

Burada div icinde olan tum p etiketlerine ulasiriz.

Genel Kardes Seciciler (General Sibling Selectors)

Ayni parent etikete sahip olan ve birbbiri ardina gelen etiketleri secmek icin kullanilir. ~ isareti ile gosterilir.

Burada dikkat edilmesi gereken iki nokta var. Birincisi p etiketi ul etiketinden sonra gelmeli (arada baska etiketler olabilir) ve ikiside ayni duzeyde yani ayni parent etiketine sahip olmalilar.

Bitisik Kardes Seciciler (Adjacent Sibling Selector)

Genel kardes seciciden tek farki belirtilen etiketler bitisik arka arkaya gelmeliler. + isareti ile gosterilir.

Sahte Sinif Seciciler (Pseudo classes)

Sahte siniflar kullanarak HTML etiketlerine CSS uygulayabiliriz. Kullanimlari selector:pseudo class {} seklindedir.

:link
Daha once tarayicida acilmamis linke CSS uygular. Not:Etkisini gormek icin tarayiciya gecmisi temizlemek gerekebilir.

:hover
Secici ile isaretlenen etiketin uzerine mouse ile gelindiginde CSS uygulanir ve mouse uzerinde degilken etkisi kaybolur.

:active
Mouse ile tiklandiginda CSS uygulanir. Tiklama kaldirildiginda etkisi kaybolur.

:first-child
Ilk child etikete CSS uygulanir.

:last-child
Son child etikete CSS uygular.

::first-latter
Blok seviyesinde bir etiketteki ilk harfe CSS uygular.

::Before
Belirtilen etiketin onune CSS uygular.

::after Belirtilen etiketin sonuna CSS uygular. Kullanimi ::before ile aynidir.
