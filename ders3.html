External, Internal ve Inline CSS Kullanimi

CSS,web sitenizin ekranda nasil gorunecegini belirleyen belirli stil kurallarina sahip dosyadir. CSS kurallari, web sitenizin HTML dosyalarina cesitli sekillerde ve yerlerde uygulanabilir. External yani harici bir still sayfasi Internal yani dahili bir stil sayfasi veya Inline yani satir ici stil kullanabilirsiniz. Her yontemin belirli kullanimlara uygun avantaji vardir.

External stil sayfasi, bir web sayfasindan baglanan bagimsiz bir .css dosyasidir. External stil sayfasinin avantaji, bir kez olusturulabilmesi ve birden cok web sayfasina uygulanabilmesidir. Site tasariminizda genis capli degisiklikler yapmaniz gerekirse eger, stil sayfasinda tek bir degisiklik yapabilirsiniz ve bu degisiklik tum baglantili sayfalara uygulanarak zamandan ve emekten tasarruf saglar.

Internal stil sayfasi, HTML dosyasinin Head bolumunde sayfa icin CSS kural kodlarini barindirir. Kural kodlari yalnizca o sayfa icin gecerlidir, ancak sayfa kodunda birden cok ogeye stil uygulamak icin kullanilabilecek class ve id'leri yapilandirabilirsiniz. Yine, CSS kodunda yapilacak tek bir degisiklik, sayfadaki tum etkilenmis ogelere uygulanacaktir.

Inline stili, belirli bir sayfa ogesine stil vermek icin,belirli bir HTML etiketi icinde kullanilir. Hizli ve kalici degisiklikler icin kullanislidirlar. ancak tasarim degisikligi yapmaya karar vermeniz durumunda, olusturdugunuz her Inline stilin ayri ayri duzenlenmesi gerektiginden, Exxternal ve Internal stil sayfalarindan daha kullanisli degildir ve digerlerine gore cok fazla tercih edilmez.

Internal CSS Kullanimi

Internal CSS, stil kodlarinin direkt HTML dosyasinin icinde head bolumunde kullanildigi bir yontemdir. head bolumunun icinde style etiketi actiktan sonra icine gerekli CSS kodlarini yazabilirsiniz.

Inline CSS Kullanimi

Inline yani satir ici stiller, dogrudan HTML kodunuzdaki herhangi bir ogeye uygulanir. Stil niteligi ve ardindan normal CSS ozellikleri bir HTML etiketi icinde belirtilir.
